# Smart File Organizer Using the Two Pointers Technique

## 1. Project Overview
A Python console application that simulates file organization tasks using the **Two Pointers technique** to perform efficient searching, pairing, partitioning, and duplicate removal.

---

## 2. Problem Statement
Develop an efficient file organizer that performs common file-processing tasks while minimizing time complexity using the Two Pointers algorithm.

---

## 3. Objectives
- Learn the Two Pointers technique.
- Implement efficient array operations.
- Compare optimized and brute-force solutions.

---

## 4. Technologies Used
- **Language:** Python
- **Data Structure:** Lists (Arrays)
- **Algorithm:** Two Pointers

---

## 5. Modules
- Display Files
- Sort Files
- Remove Duplicates
- Find Pair
- Find All Pairs
- Closest Pair
- Pair Smallest & Largest
- Partition Files
- Remove Large Files
- Compress Duplicates
- Maximum Container
- Brute Force vs Two Pointers Comparison

---

## 6. Two Pointer Patterns
- **Slow & Fast Pointers:** Duplicate removal, partitioning, compression.
- **Opposite-End Pointers:** Pair searching, closest pair, maximum container.

---

## 7. Learning Outcomes
- Apply different Two Pointer patterns.
- Optimize array-based problems.
- Improve algorithm analysis and problem-solving skills.

---

## 8. Conclusion
This project demonstrates practical applications of the **Two Pointers technique**, showing how optimized algorithms improve efficiency compared to traditional approaches.
