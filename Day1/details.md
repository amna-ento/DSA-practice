# Product Inventory Management System 

## Problem Statement

Develop a simple **Product Inventory Management System** for a warehouse using arrays. The system should allow warehouse staff to:

- Load product data from a file.
- Search for a product using its Product ID.
- Insert a new product into the inventory.
- Sort products in ascending order of Product ID.
- Generate an updated warehouse inventory report.

The implementation should focus on applying fundamental Data Structures and Algorithms concepts without relying on built-in sorting methods.

---

## DSA Topics Covered

- Arrays
- Array Traversal
- Linear Search
- Array Insertion
- Sorting (without built-in `sort()`)
- File Handling
- Functions / Modular Programming
- Time Complexity Analysis (Basic)
